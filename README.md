#ZiLaVue

sekedar percobaan penggunaan laravel dan Vue Js2


## Instalasi
###### 1. clone repository
```bash
git clone https://github.com/faozimipa/zilavue.git Nama_Foldermu
```
###### 2. install vendor 
```bash
composer install
```
###### 3. add package
buka package.json tambahkan 
```bash

    "vue-router": "^2.0.0",
    "vue-strap": "^1.1.28"

```
pada devDependencies.
###### 4. install npm
```bash
npm install
```
